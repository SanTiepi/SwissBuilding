Create a composite geospatial risk score combining all 24 geo.admin layers into a single 0-100 score with breakdown.

- [ ] Integrated into enrichment pipeline

## Technical Details

File: `backend/app/services/enrichment/score_computers.py`
Already has: neighborhood_attractiveness_score, livability_score, environmental_risk_score

**New score formula:**
```python
geo_risk = weighted_sum(
    flood_risk * 3.0,
    seismic_risk * 2.5,
    contamination * 3.0,
    radon * 2.0,
    noise * 1.5,
    landslide * 2.0,
    rockfall * 2.0,
    groundwater_restriction * 1.0,
    seveso_proximity * 2.5,
) / total_weights
```
