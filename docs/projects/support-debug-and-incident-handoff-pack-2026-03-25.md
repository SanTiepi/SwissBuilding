# Support Debug and Incident Handoff Pack

Date de controle: `25 mars 2026`

## Purpose

When something goes wrong in production-like use, SwissBuilding should make it
easy to hand the case from operator to support or engineering without losing
context.

## Rule

Support handoff should preserve product context, not just technical error text.

## Focus

- affected building or workflow
- current blocker or failure mode
- recent relevant events
- known environment state
- reviewer or engineer handoff context

## Acceptance

This pack is succeeding when support and engineering can resume a case without
reconstructing the story manually.
