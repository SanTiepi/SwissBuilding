# Case Study Template and Proof Story Pack

Date de controle: `25 mars 2026`

## Purpose

SwissBuilding should turn successful workflows into reusable proof stories.

## Rule

A case study should be workflow-centered, not slogan-centered.

## Template

- starting pain
- workflow used
- blocker or ambiguity removed
- proof or pack reuse created
- measurable improvement
- quote or operator reaction if available

## Acceptance

This pack is succeeding when successful pilots and slices become reusable proof
stories for future buyers.
