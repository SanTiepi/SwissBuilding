- treat these files as executable briefs, not long-term strategy
- `MEMORY.md` holds durable decisions
- `docs/product-frontier-map.md` holds the broadest idea bank
- `docs/roadmap-next-batches.md` holds long-range structure
- `ORCHESTRATOR.md` decides what is active now
- if a new brief appears, assign it to one macro-domain and one ring immediately
