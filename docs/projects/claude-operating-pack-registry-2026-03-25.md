# Claude Operating Pack Registry

Date de controle: `25 mars 2026`

## Purpose

This registry is the shortest way to navigate the operating packs and briefs
without rereading the whole strategy corpus.

Use it when one of these becomes true:

- there are too many packs to remember
- a next wave must be chosen quickly
- a brief must be copied without strategic re-triage
