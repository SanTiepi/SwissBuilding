## What G2 Should Include

### 1. Prospect-Grade Seed Simplification

The current seeded/demo reality is rich, but may still be too encyclopedic for commercial use.

G2 should create one prospect-grade scenario that is:

- readable
- credible
- not overloaded
- anchored in the wedge

This does **not** mean deleting the richer seed universe.
It means creating one cleaner demo/pilot path on top of it.

---

### 2. Pilot Baseline and Scorecard

The scorecard should measure a very small set of operationally meaningful metrics, such as:

- dossier completeness
- readiness state
- missing proof count
- blocker count
- manual correction count
- pack generation readiness
- time-to-safe-to-start where available

The product should support a clear before/after or baseline/progress framing.

Rule:
- no vanity metrics
- no giant dashboard
- no analyst theater

---

### 3. Operator Loop

The daily/weekly operator flow during a pilot should feel explicit and lightweight.

Minimum desired loop:

1. open the pilot scope
2. see which dossiers/buildings are blocked or incomplete
3. see what is the next useful action
4. resolve or assign the next missing item
5. regenerate/reassess readiness
6. produce or refresh the authority-ready output

This may use:

- `Today`
- `Building Home`
- `Case Room`
- minimal pilot summary surfaces

---

### 4. Pilot-Ready Demo Path

The product should support a short demo path aligned with the commercial script:

- here is the building/case
- here is why it looks documented but is not truly ready
- here is what is missing / contradictory / stale
- here is the next action path
- here is the authority-ready pack path
- here is the evidence/provenance backing the verdict

This should feel closer to:

- a reliable proof demo

than to:

- a huge product tour

---

### 5. Minimal Client-Facing Pilot Readiness Support

Only what directly supports pilot execution:

- champion-facing clarity
- sponsor-facing summary
- pilot metric visibility
- clean baseline/progress narrative

Do not build:

- a broad account management system
- generic admin analytics
- broad sales CRM features

---
