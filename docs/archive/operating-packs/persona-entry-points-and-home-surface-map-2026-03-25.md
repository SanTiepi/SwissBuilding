# Persona Entry Points and Home Surface Map

Date de controle: `25 mars 2026`

## Purpose

SwissBuilding should not feel like the same home page for everyone.

Different users need different primary entry points into the same canonical
truth.

## Rule

Each persona should have one obvious home surface and one obvious weekly review
surface.

## Persona map

### Property manager

Primary home:

- ControlTower

Weekly review:

- portfolio view

### Owner

Primary home:

- building overview

Weekly review:

- proof and readiness summary

### Procedure coordinator

Primary home:

- procedure queue

Weekly review:

- authority request and blocker review

### Public owner

Primary home:

- portfolio ControlTower

Weekly review:

- committee or governance packs

### Diagnostician using Batiscan and SwissBuilding together

Primary home:

- Batiscan for execution
- SwissBuilding continuity surface for publication and downstream use

Weekly review:

- publication and reuse outcomes

## Product rule

Do not make every persona start from the same generic dashboard.

## Acceptance

This map is succeeding when a new user can tell:

- where to start
- what to check weekly
- how that connects back to canonical building truth
