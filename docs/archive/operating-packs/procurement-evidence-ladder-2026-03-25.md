# Procurement Evidence Ladder

Date de controle: `25 mars 2026`

## Purpose

Public and enterprise buyers often need stronger evidence before they trust a
new operating layer.

This ladder defines what evidence SwissBuilding should be able to show as buyer
trust requirements rise.

## Ladder

### Level 1 - product legibility

Need to show:

- clear workflow value
- understandable surface
- no obvious confusion

### Level 2 - operational proof

Need to show:

- recurring use
- visible blockers and next actions
- reduced dossier rebuild work

### Level 3 - procedural proof

Need to show:

- route clarity
- request and response trace
- review and confidence semantics

### Level 4 - proof defensibility

Need to show:

- source
- version
- freshness
- delivery or acknowledgement trace

### Level 5 - governance and auditability

Need to show:

- decision trace
- source-of-truth boundaries
- escalation and review handling

## Acceptance

The ladder is succeeding when the team can match a buyer's trust threshold to
visible product proof instead of generic reassurance.
