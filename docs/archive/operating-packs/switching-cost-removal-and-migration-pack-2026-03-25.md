# Switching Cost Removal and Migration Pack

Date de controle: `25 mars 2026`

## Purpose

SwissBuilding will not spread only because it is better.

It must also be easier to adopt than the status quo is to maintain.

This pack defines the product behaviors that reduce switching cost.

## Rule

Every major feature should help with at least one of:

- less migration pain
- less retraining
- less re-entry of existing truth
- easier coexistence with incumbent tools

## Main switching costs

### Historical memory loss

Fear:

- if we switch, we lose context

Product answer:

- timeline
- imports
- handoff package
- source provenance

### Workflow disruption

Fear:

- the team cannot stop operations to learn a new system

Product answer:

- operating inbox first
- partial adoption by workflow
- coexistence with ERP and chantier tools

### Trust loss

Fear:

- official or important proofs will be less defensible

Product answer:

- proof delivery
- confidence ladder
- visible provenance
- explicit caveats

### Re-entry fatigue

Fear:

- same data has to be retyped

Product answer:

- imports
- intake conversion
- exchange contracts
- once-only evidence reuse

## Product patterns that lower switching cost

### Land-and-expand

Start with one habit:

- ControlTower
- packs
- diagnostics publication
- procedure management

Do not require full replacement day one.

### Coexistence first

Product should say:

- keep your ERP
- keep your chantier tool
- keep your current inboxes briefly
- centralize what matters first

### Progressive migration

Migrate by:

- workflow
- building cluster
- audience
- proof package type

Not always by big-bang tenant cutover.

### Import with provenance

Imported truth should preserve:

- source system
- import date
- original identifier
- confidence and caveat when mapping is imperfect

## Acceptance

This pack is succeeding when the product can honestly say:

- adopt one slice without replacing everything
- preserve old truth while building new truth
- migrate gradually without losing continuity
